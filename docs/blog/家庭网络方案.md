## Alist

> 挂载本地：`  -v /home/spa/alist_localstorage:/mnt`

```dockerfile
docker run -d \
  --restart=always \
  -v /etc/alist:/opt/alist/data \
  -v /home/spa/alist_localstorage:/mnt \
  -p 5244:5244 \
  -e PUID=0 \
  -e PGID=0 \
  -e UMASK=022 \
  --name="alist" \
  xhofe/alist:latest
```
