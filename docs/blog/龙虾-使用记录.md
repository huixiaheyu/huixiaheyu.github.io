Gateway 网关是会话、路由和渠道连接的唯一事实来源。

## 启动

安装：

```js
// 更推荐npm这种方式，便于管理
npm install -g openclaw@latest
openclaw onboard --install-daemon
```

重启：

```js
openclaw gateway status   # 查看状态
openclaw gateway start    # 启动
openclaw gateway stop     # 停止
openclaw gateway restart  # 重启
```
